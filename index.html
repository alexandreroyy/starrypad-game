Usage proposé

Hypothèse d'utilisation de matériel informatique plausible

L'idée est d'utiliser le Donner StarryPad (contrôleur MIDI à 16 pads lumineux) comme support d'un jeu de mémoire basé sur les lumières. Je veu faire une sorte de jeu comme "Simon Says".

Déroulement d'une partie

  • L'utilisateur lance une partie en appuyant sur un bouton dédié.
  • Une séquence de pads s'allume successivement (couleurs/lumières). La séquence est aléatoire.
  • À la fin de la démo, l'utilisateur doit reproduire la même séquence en appuyant sur les mêmespads dans le bon ordre.
  • La difficulté peut augmenter progressivement (longueur de la séquence, vitesse, etc.).

Description de l'objet

Résumé des fonctionnalités

La fonctionnalités de base de ce pad est de l'utiliser avec un logciel comme GarageBand pour pouvoir jouer des notes et ensuite les enregistrer pour créer de la musique.

Coté hardware, ce pad est composé de:

  • 16 pads en grille 4x4
  • Rétroéclairage RGB
  • Connexion USB-C à USB-A

Fabricant(s)

Donner Music

Autres éléments pertinents

Le StarryPad peut être piloté depuis un ordinateur avec Python grâce à des librairies MIDI (par exemple mido ou rtmidi), ce qui permet d'écrire des scripts pour lire les appuis sur les pads et contrôler leurs lumières.
